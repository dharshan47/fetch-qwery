# 🧪 Fetch-Qwery Demo App

This directory contains a Next.js playground demonstrating the capabilities of the `fetch-qwery` package.

## Features Showcased

Explore the `app/examples/` directory to see `useFetch` in action:
- **SSR & Hydration:** Safe data fetching across Server and Client components.
- **Localhost Optimization:** Fast revalidation skips during development.
- **Image Preloading:** Automatic `Loading=eager` header generation for image assets.
- **Any API Integration:** Standard data fetching flows with robust error handling.

## Running Locally

To start the demo playground:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to interact with the test applications.
