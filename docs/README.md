# 🚀 Fetch-Qwery Documentation

This directory contains the official documentation website for the `fetch-qwery` package. It is built using [Next.js](https://nextjs.org) and [Fumadocs](https://github.com/fuma-nama/fumadocs).

## Quick Start

To run the documentation site locally:

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

The documentation site is designed to be easily deployed to [Vercel](https://vercel.com/new).

When importing the GitHub repository into Vercel, ensure you set the **Root Directory** to `docs` in the project settings.
