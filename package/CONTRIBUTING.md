# Contributing to Fetch Qwery

First off, thanks for taking the time to contribute! 🎉

## Development Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Run tests: `npm test`
4. Run demo: `npm run dev`

## Pull Request Process

1. Create a new branch for your feature or bugfix.
2. Ensure tests pass and the code follows the existing style.
3. Submit a PR describing your changes.

## Code of Conduct

Please be respectful and helpful to others in the community.
